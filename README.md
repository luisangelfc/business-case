#  Sistema de Candidatos – Business Case (Full Stack Jr)

Aplicación web desarrollada en **React** que permite visualizar, buscar, filtrar y ordenar candidatos de forma eficiente.  
Este proyecto fue realizado como **business case técnico** para un proceso de selección en un entorno de startup.

---

## Vista general

La aplicación incluye:

- Listado de candidatos
- Búsqueda por nombre o email
- Filtros dinámicos
- Ordenamiento por fecha de registro
- Estados visuales claros (RFC, migrante, experiencia)
- Interfaz responsive y moderna

---

##  Objetivo del proyecto

Construir una interfaz clara y funcional que simule un **sistema interno de gestión de talento**, enfocándose en:

- Buen manejo de estado en React
- Lógica de filtrado eficiente
- Código legible y mantenible
- Experiencia de usuario clara
- Preparación para futura integración con APIs, chatbots o agentes conversacionales

---

## Tecnologías utilizadas

- **React** (componentes funcionales y hooks)
- **JavaScript (ES6+)**
- **Tailwind CSS**
- **Lucide React** (íconos SVG)
- **PostCSS**
- **Mock data (JSON)** para simular backend
